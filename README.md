# Shopping cart
Uma criação de um simples sistema de busca e carrinho de compras.

# Tecnologias usadas:
- RHTML
- Javascript
- CSS

# Instalação
- Clone o repositório: "git clone https://github.com/ThiagoMarchini/shopping-cart.git"
